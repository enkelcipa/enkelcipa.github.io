Project Interesting Mundane Tasks

Project Interesting Mundane Tasks (IMT) is a job simulator game made in Unreal Engine 5.6 for my Final Year Project at University. This game is based around the question of "How do I make mundane tasks interesting and engaging?".

This project allowed me to explore how developers retain player's attention when the task at hand is uninteresting and boring.
