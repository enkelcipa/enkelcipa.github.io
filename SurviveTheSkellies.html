Survive the Skellies

Survive the Skellies is a Survivors-like game built in Unreal Engine 5.6 for my C++ for Designers unit at University. The goal was to use C++ in Unreal Engine to create a designer friendly game, with classes that could be used to create actors with functionality in blueprint. I designed and implemented systems for a survivors-like game, weapons, enemy spawning in waves, EXP and level up systems and stat scaling.
