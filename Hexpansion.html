Hexpansion

Hexpansion is a city builder game built in Unreal Engine 5.6 for my Prototyping for Technical Designers unit at University. We were tasked to pick a genre from a list, then produce a visually and technically polished game within 8 weeks.

Hexpansion features a hexagonal grid map with procedurally generated biomes. I designed and implemented systems for building placement, destruction and a responsive road placement system that automatically adjusts to the correct mesh and orientation.

Procedurally Generated Map

The map in Hexpansion is always constructed to be the same size. This value can be set by the developer/designer. I use procedural generation to generate biomes with the use of Perlin Noise. This value equates to 1 of 4 biomes in the game.

This is what the first pass of the generation looks like before the biomes are expanded and cleaned up.

Once the biomes are expanded we can see that they take up larger portions of the map, but we do occasionally see a tile that is on its own and has no neighbouring tiles of the same biome.

I then run my final pass to remove all the single tiles. From this image we can see that every tile has at least 1 neighbour that is of the same biome.
Ideally I would do another pass which removes all the small biomes (2-3 tiles) but I was constrained by time.

Dynamic Road Placement

This was something that also took a considerable amount of time in my project.
When a player places a road, the tile looks at all the neighbouring tiles to see if they are also roads. The tile then selects the correct road mesh to use and rotate it accordingly.

I decided that using a bitmask was the best approach to handle road placement. Each road type has its own bitmask value assigned to it.
I then calculate which road I need based off of the neighbouring road tiles and rotate the mesh accordingly.

Building Placement and Destruction

When Build Mode is toggled, the player can select buildings in the UI. This then spawns the building with a transparent material that alternates between red and green to indicate whether it can be placed or not.
